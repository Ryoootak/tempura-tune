# TempuraTune

スマホのマイクで油の音を聴き、AIが温度を判定するアプリ。
日本の天ぷら職人の技をAIで再現する。

## コンセプト
温度計を持っていなくても、スマホをキッチンに置いて濡れた
箸を油に入れるだけで、適温かどうかが分かる。
