# Phase 4: 準リアルタイム録音UI実装

## 目的
`design/TempuraTune.html` をベースに、2秒録音を繰り返して
結果を更新する準リアルタイムUIを実装する。

## 方針
- 完全ストリーミングは採用しない
- `MediaRecorder` で約2秒ごとに音声を切り出す
- 各チャンクを `/api/analyze` に送信する
- 直近の判定結果でメーターと文言を更新する
- MVPでは「2秒録音 → 送信 → 更新 → 次の2秒録音」を順番に回す

## 実装メモ
- `src/app/page.tsx` に MVP の料理選択画面と測定画面を実装
- `MediaRecorder` を使って約2秒ごとに録音し、`/api/analyze` に順次送信
- 最新結果をメーター、ゾーン表示、確信度、アドバイスへ反映
- 権限未許可、API失敗、ブラウザ非対応時はエラー表示と再試行導線を出す
