# Phase 5: UI改善

## 目的
準リアルタイム判定の見やすさと安心感を高め、スマホで使いやすい
体験に仕上げる。

## 方針
- 判定待ち時間を退屈に見せない
- 最新判定と直前状態の差が分かるようにする
- 調理中でも読みやすいコントラストと大きさを優先する

## タスク
- [ ] メーター更新アニメーション
- [ ] 録音中の脈動や波形風演出
- [ ] スマホ縦向き最適化
- [ ] 結果文言の視認性改善
- [ ] アクセシビリティ改善

## 完了条件
- [ ] 判定の状態変化が見て分かる
- [ ] スマホ縦画面で無理なく操作できる
- [ ] 高コントラストでも破綻しない
